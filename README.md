# AIPI590_Assignment2
This is a repository for creating an adversarial patch. 

### Important documents include:
- Assignment2_Adversarial_Patch.ipynb: a Jupyter notebook including all the documented codes and explanation for creating the adversarial patch
- imagenet_classes.txt file: all the classes that can be classified by the Torchvision ResNet34 model 
- custard_apple.jpg & banana.jpg: input images that are chosen from the class in imagenet_classes.txt file
- merge_image.pdf: output adversarial patch after conducting adversarial changes (merge)
